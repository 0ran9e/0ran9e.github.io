Implementation

Warmup exercise：sigmoid function

在开始解出实际的成本函数之前，回忆一下逻辑回归的假设的定义为：
$$
h_{θ}(x) = g(θ^{T}x)
$$
函数g是 sigmoid函数，它的定义为：
$$
g(z) = \frac{1}{1+e^{-z}}
$$
显而易见的是这个函数在后面将会一直被用到，所以我们第一步就是现在 sigmoid.m 中实现它。完成后可以自己调用 sigmoid(x) 来测试一下，对于一个大的正数，sigmoid应该趋近于1，而对于一个大的负数，sigmoid应该趋近于0。当x为0时，sigmoid的值为0.5。你的代码也应该支持向量与矩阵，对于矩阵，你的函数应该在每个元素上执行sigmoid函数。

这题我本来的用的是两个循环分别表示行与列，进行遍历将每个元素执行sigmoid函数…但是后来看到了这种点运算的方法，代码简洁优美多了，所以我就po这种方法的啦。

Cost function and gradient

如果前面sigmoid函数代码没有问题的话，我们就可以用它来完成逻辑回归的成本函数以及其梯度的计算。如果你把线性回归的成本函数直接用作逻辑回归的成本函数的话，你会发现此时的成本函数图形显示被称为”非凸函数“，梯度下降算法就不能用了。事实上，逻辑回归的成本函数是这样的：
$$
J(θ) = \frac{1}{m}\sum^{m}{i=1}{[-y^{(i)}log(h{θ}(x^{(i)}))-(1-y^{(i)})log(1-h_{θ}(x^{(i)}))]}
$$
成本的梯度是一个与 θ 长度相等的向量，其中 $j^{th}$ 元素的定义如下：
$$
\frac{∂J(θ)}{∂θ{j}} = \frac{1}{m}\sum^{m}{i=1}{(h_{θ}(x^{(i)})-y^{(i)})x^{(i)}{j}}
$$
这个梯度公式看起来与线性回归梯度一摸一样，但实际上是不同的，因为它们的 $h{θ}(x)$ 不一致的。

Learning parameters using fminunc

在之前的作业中，我们利用梯度下降算法找出了了线性回归模型的最佳参数 θ ，我们编写了一个成本函数并计算了梯度，然后在迭代中进行梯度下降。在这一节我们利用MATLAB的内置函 fminunc 更加高效的做到同样的作用。

在这题中，我们希望利用 fminunc 函数来得到使逻辑回归的成本最优化的参数 θ 。具体来说，当我们使用这个函数时，我们一般需要传递三个参数：

  • 我们试图优化的参数的初始值
  • 一个当我们给定训练集和特定的 θ 时，能够计算逻辑回归成本和梯度的函数
  • 还有这个函数的可选项，这个因为比较常见，所以是我自己补充上的

　　在前面练习中，我们已经得到了这个需要的函数，所以就可以直接拿来用了

解释一下参数 options 的意思，将 GradObj 选项置 on 告诉了 fminunc 我们的函数返回成本和渐变，所以 fminunc 在最小化的时候就可会用到渐变。MaxIter 设置为400意为在 fminunc 终止前最多运行400次。（说实话...@的意思我看了作业文档还是不太懂…我觉得或许类似于C++中的函数指针？？？以后学到了再回来填坑吧…)

如果一切都正确，那么 fminunc 就将返回最优化后的成本和 θ 最终值。所以利用这个 fminunc ，我们只需要完成计算成本与梯度的函数就行了。而且这个函数的性能也比我们自己写的循环的那种梯度下降算法要优秀的多。

这个最终的 θ 值也会被用来绘制训练集上的决策边界(Decision Boundary)，绘图的函数作业已经帮我们写好了，看一看学习一波就好

Visualization the data

与之前一部分练习相同，我们用 plotData( ) 来画出图像。轴是两个测试的分数，正数(y=1)表示质量可接受，负数(y=-1)相反。

Feature mapping

为了更好的适应数据分布，我们可以为每个数据点创建更多的特征。我们将把特征映射到x1和x2的所有多项式，直到6个幂。在 mapFeature.m 中，也已经给出了实现，因为前面画图的函数也有用到，所以在前面就给出代码了。
$$
mapFeature(x) = \begin{bmatrix}
1\
x_{1}\
x_{2}\
x_{1}^{2}\
x_1x_2\
x_{2}^{2}\
x_{1}^{3}\
.\.\
x_{1}x_{2}^{5}\
x_{2}^{6}
\end{bmatrix}
$$
通过这种映射，我们两个特征的向量就被转换成28维向量。在这个高维特征向量上训练的逻辑回归将会具有更复杂的决策边界，我们在二维图中绘制时就将呈现非线性的图形。

Cost function and gradient

现在我们要完成实现计算正则化逻辑回归的成本函数和梯度的代码。

逻辑回归的成本函数定义如下：
$$
J(θ) = \frac{1}{m}\sum_{i=1}^{m}[-y^{(i)}log(h_{θ}(x^{(i)}))-(1-y^{(i)})log(1-h_{θ}(x^{(i)}))] + \frac{λ}{2m}\sum_{j=1}^{n}θ{j}^{2}
$$
需要注意的是，参数 $θ{0}$ 不需要调整。因为MATLAB中索引是从1开始的，也就是说不需要调整theta(1)的值。

成本函数的梯度是一个向量，其中 $j^{th}$ 的元素定义如下：
$$
\frac{∂J(θ)}{∂θ{0}} = \frac{1}{m}\sum{i=1}^{m}(h_{θ}(x^{(i)})-y^{(i)})x_{j}^{i} \qquad for \ j=0
$$

$$
\frac{∂J(θ)}{∂θ{0}} = \frac{1}{m}\sum{i=1}^{m}(h_{θ}(x^{(i)})-y^{(i)})x_{j}^{i}+\frac{λ}{m}θ_{j} \qquad for \ j ≥ 1
$$
